== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Conexao G Elementor'
3. Activate Conexao G Elementor from your Plugins page.

=== Manually ===

1. Upload the `conexaog-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the Conexao G Elementor plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
